### ajche dotfiles
tested on arch, with niri
managed with stow

### Packages:
- Neovim
- tmux
- kitty
- fish
- niri
- rofi
- taskwarrior

### Dependencies:
- Starship
- fzf
- ripgrp
- tpm

### Instructions:
**\[tmux]**
download tpm:
> git clone https://github.com/tmux-plugins/tpm ~/.local/share/tmux/plugins/tpm

**\[fish]**
download starship:
> curl -sS https://starship.rs/install.sh | sh
